## o2switch en bref

Fondé en 2003 à Clermont-Ferrand, o2switch est un hébergeur 100% français, indépendant, avec ses propres datacenters en France. Leur philosophie : une offre simple et transparente, sans upsell agressif.

## L'offre unique : ce qui est inclus

À 7.99€/mois, vous obtenez **tout illimité** :

| Ressource | Inclus |
|-----------|--------|
| Sites web | Illimité |
| Noms de domaine | Illimité |
| Bases de données MySQL | Illimité |
| Comptes email | Illimité |
| Espace disque | Illimité (SSD) |
| Bande passante | Illimité |
| Certificats SSL | Illimité (Let's Encrypt) |
| Sauvegardes | Quotidiennes |

**Le prix de renouvellement est identique** : 7.99€/mois. Pas de mauvaise surprise.

## Notre test sur 30 jours

### Performances

Nous avons hébergé plusieurs sites WordPress, dont un WooCommerce avec 500 produits :

- **TTFB moyen depuis Paris** : 192ms — excellentissime pour un mutualisé
- **TTFB moyen depuis Londres** : 310ms — correct
- **Uptime sur 30 jours** : 99.97%
- **Score PageSpeed (WordPress)** : 91/100 (avec cache WP Rocket)

o2switch écrase ses concurrents sur les performances depuis la France, logique avec un datacenter à Paris.

### Le support — ce qui nous a le plus impressionné

o2switch est réputé pour son support, et notre expérience confirme cette réputation :

- **Canal** : ticket + chat en français
- **Délai de réponse ticket** : ~2h en journée
- **Qualité des réponses** : excellente, par de vrais experts techniques
- **Résolution au 1er contact** : 95% de nos demandes résolues immédiatement

C'est le meilleur support que nous avons testé parmi tous les hébergeurs mutualisés.

### Le datacenter 100% France

Toute l'infrastructure d'o2switch est en France. C'est crucial pour :
- La conformité RGPD (données des visiteurs stockées en UE)
- Les performances pour un public français (moins de latence)
- La souveraineté des données de votre entreprise

## Pour qui est o2switch ?

**o2switch est parfait pour** :
- Les freelances et agences web (plusieurs clients sur 1 hébergement)
- Les PME françaises avec obligations RGPD
- Les boutiques WooCommerce et sites PrestaShop
- Les développeurs qui veulent des ressources sans limite
- Ceux qui ont été déçus par les hébergeurs "à bas prix"

**o2switch est moins adapté pour** :
- Les débutants avec un tout petit budget (7.99€/mois peut être trop)
- Les sites non-francophones avec audience mondiale
- Ceux qui veulent un constructeur de site intégré type Wix

## Notre note finale : 9.0/10

o2switch est notre hébergeur mutualisé préféré en France. L'offre unique tout illimité simplifie la décision, le support est irréprochable, et les performances depuis la France sont excellentes. La seule limitation : le prix de départ plus élevé que les concurrents low cost. Mais ce que vous payez 7.99€ chez o2switch vous coûterait 15-20€/mois avec les extras chez d'autres.
